# @ui5-editor-tools/logic-utils

Common shared logic related utilities for other packages in this mono-repo.

## Installation

With npm:

- `npm install @ui5-editor-tools/logic-utils`

With Yarn

- `yarn add @ui5-editor-tools/logic-utils`
